About

Passionate about chemistry, I devoted considerable time to finding an idea for my first project that would not only match my skill level but also be both exciting and challenging. The concept of merging chemistry with programming stood out as a fantastic and stimulating choice. Thus, I decided to create an interactive periodic table based on an API, aiming to assist chemistry students.
ElemLearn is not only a valuable learning tool but also a Portfolio Project for Holberton School, showcasing the integration of technical skills with a passion for education.
